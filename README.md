# StayCalm

StayCalm is a minimal iOS app designed to provide continuous ambient sound environments for calm, focus, and relaxation.

The app works fully offline and does not require user accounts, registrations, or data collection.

## Features
- Continuous ambient sound playback
- Simple and distraction-free interface
- No accounts, no tracking, no analytics
- Works offline
- Background audio support
